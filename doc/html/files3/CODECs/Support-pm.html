encode_tlv_parameters

sub encode_tlv_parameters

Encode the tlv parameters of a TLV message.  This method as a structured message parameter, it contains the values to encode with one entry for each field.  It has a parameters descriptor table which determines the type and encoding options for each field and a message descriptor which contains the list of acceptable fields.

Parameters

$msgStructured message
$tag_lengthlength of the command tag
$len_lengthlength of the command lenght field
$params_descreference to tlv parameters descriptors
$reversereversed tlv parameters descriptor reference
$msgs_descmessage descriptor

Returns: a binary buffers containing the tlv parameters
