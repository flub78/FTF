FileSets.pm
Version1.0

Abstract

Manages lists with unique elements.  Addition of an element already present in the list does not change it.

Curently, it is just a set of routines to handle a regular Perl list.  Perhaps that it would be more convenient to make it a real class with addition, merge, intersection, operator overloading, etc.
